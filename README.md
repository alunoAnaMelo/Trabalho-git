# Mundo em Foco v1.0

Projeto desenvolvido para a atividade de Git e GitHub.

## Sobre o projeto

O Mundo em Foco é um site que reúne informações sobre diferentes países do mundo. Cada integrante da equipe ficou responsável por pesquisar e desenvolver uma página sobre um país específico.

## Países abordados

* Japão
* México
* Alemanha
* Itália

## Tecnologias utilizadas

* HTML5
* CSS3
* Git
* GitHub
* GitHub Pages

## Integrantes

* Mariana – Japão
* Ana Luísa – México
* Raphael Braga – Alemanha
* Giovanni – Itália

## Versionamento

Durante o desenvolvimento do projeto foram utilizadas branches individuais, commits, pull requests, merges e resolução de conflitos, permitindo a integração das páginas produzidas por cada integrante.
